# ⚕️ Healpoint - Doctor Booking (MERN)

A full-featured doctor appointment booking platform for patients, doctors, and administrators. Built with the MERN stack (MongoDB, Express, React, Node.js), this application enables seamless health appointment scheduling and management.

## 🌐 Live Demo

- **User (Patient) App:** [https://doctor-booking-mern-frontend.onrender.com/](https://doctor-booking-mern-frontend.onrender.com/)
- **Admin & Doctor Panel:** [https://doctor-booking-mern-admin.onrender.com/](https://doctor-booking-mern-admin.onrender.com/)

## ✨ Features

### 🧑 User (Patient)
- Register and login
- Browse and filter doctors
- Book appointments
- View upcoming appointments, cancel or reschedule
- Update profile and personal informations

### 👨‍⚕️ Doctor
- Doctor dashboard (Earnings, Total Patients, Total Appointments, Latest Appointments)
- Manage appointment requests (approve or cancel)
- Access all details about appointments and patients
- View and update profile

### 🛂 Admin
- Admin dashboard (Total Doctors, Total Patients, Total Appointments, Latest Appointments)
- Manage doctor registrations
- Access all details about appointments and doctors
- Handle platform moderation and system analytics

---

## 🧱 Tech Stack

### Frontend
- **React.js** – Core library for UI development
- **Tailwind CSS** – Utility-first CSS framework for styling
- **React Router DOM** – Declarative routing and navigation
- **Context API** – Global state management
- **Axios** – Promise-based HTTP client for API calls
- **React Toastify** – Elegant toast notifications

### Backend
- **Node.js** & **Express.js** – Backend framework and REST API
- **MongoDB** with **Mongoose** – NoSQL database and ORM
- **JWT (JSON Web Token)** – Secure authentication and authorization
- **Multer** – File upload handling (e.g., documents, profile images)
- **Cloudinary** – Cloud-based image storage and management
- **bcrypt** – Password hashing for security
